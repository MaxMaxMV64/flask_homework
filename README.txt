# Календарь

Это программа RESTful API для управления событиями в календаре. Программа позволяет выполнять CRUD-операции с событиями, есть ограничения на максимальную длину заголовков и текстов.

# Основные требования к IDE
- Python 3.6+
- Flask 3+
- TinyDB

# Запуск программы:

python main.py
